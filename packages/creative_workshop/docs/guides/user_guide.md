# Creative Workshop 用户指南

## 概述

Creative Workshop 是一个功能强大的创意工坊模块，为用户提供绘画、游戏、项目管理等多种创意功能。本指南将帮助您快速上手并充分利用这些功能。

## 快速开始

### 1. 安装和配置

在您的 Flutter 项目中添加 Creative Workshop 依赖：

```yaml
dependencies:
  creative_workshop:
    path: ../packages/creative_workshop
```

### 2. 基本使用

```dart
import 'package:creative_workshop/creative_workshop.dart';

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        body: CreativeWorkspace(),
      ),
    );
  }
}
```

## 主要功能

### 1. 工作区界面

Creative Workshop 的主界面包含以下几个主要区域：

#### 1.1 工具栏 (Toolbar)
- 位置：通常在左侧或顶部
- 功能：选择不同的绘画工具
- 工具类型：
  - 画笔工具：用于自由绘画
  - 铅笔工具：用于精细绘制
  - 形状工具：绘制几何形状
  - 橡皮擦：擦除内容
  - 选择工具：选择和移动对象

#### 1.2 画布区域 (Canvas)
- 位置：界面中央
- 功能：主要的绘画和创作区域
- 特性：
  - 支持多点触控
  - 压感支持
  - 无限画布
  - 缩放和平移

#### 1.3 属性面板 (Properties Panel)
- 位置：通常在右侧
- 功能：调整当前工具的属性
- 可调整属性：
  - 颜色选择
  - 画笔大小
  - 透明度
  - 混合模式

#### 1.4 项目浏览器 (Project Browser)
- 位置：可停靠面板
- 功能：管理和浏览项目
- 操作：
  - 创建新项目
  - 打开现有项目
  - 删除项目
  - 导入/导出项目

#### 1.5 状态栏 (Status Bar)
- 位置：界面底部
- 功能：显示当前状态信息
- 信息包括：
  - 当前工具
  - 画布坐标
  - 缩放级别
  - 项目状态

### 2. 绘画功能

#### 2.1 基本绘画

1. **选择工具**：在工具栏中点击所需的绘画工具
2. **调整属性**：在属性面板中设置颜色、大小等参数
3. **开始绘画**：在画布上拖动鼠标或手指进行绘画

#### 2.2 高级绘画技巧

**图层管理**：
- 创建新图层：右键画布 → "新建图层"
- 切换图层：在图层面板中点击图层
- 调整图层顺序：拖拽图层重新排序

**颜色管理**：
- 使用调色板：点击属性面板中的颜色选择器
- 自定义颜色：使用RGB或HSV滑块调整
- 颜色历史：查看最近使用的颜色

**画笔设置**：
- 大小调整：使用滑块或输入具体数值
- 硬度设置：调整画笔边缘的柔和程度
- 流量控制：控制颜料的流动速度

#### 2.3 形状绘制

1. 选择形状工具（矩形、圆形、线条等）
2. 在画布上拖拽定义形状大小
3. 在属性面板中调整填充和描边属性

### 3. 项目管理

#### 3.1 创建项目

1. 点击"文件" → "新建项目"
2. 选择项目模板：
   - 绘画项目：适合自由绘画
   - 设计项目：适合图形设计
   - 游戏项目：适合游戏开发
   - 动画项目：适合动画制作
3. 设置项目名称和描述
4. 点击"创建"完成

#### 3.2 保存和加载

**自动保存**：
- 系统每5分钟自动保存一次
- 可在设置中调整自动保存间隔

**手动保存**：
- 快捷键：Ctrl+S (Windows/Linux) 或 Cmd+S (Mac)
- 菜单：文件 → 保存

**加载项目**：
- 在项目浏览器中双击项目
- 或使用"文件" → "打开项目"

#### 3.3 项目导出

1. 选择"文件" → "导出"
2. 选择导出格式：
   - PNG：适合图像
   - SVG：适合矢量图形
   - PDF：适合文档
   - JSON：项目数据格式
3. 设置导出参数
4. 选择保存位置

### 4. 游戏功能

#### 4.1 切换到游戏模式

1. 点击工作区顶部的"游戏"标签
2. 或使用快捷键 Ctrl+G

#### 4.2 内置游戏

**点击游戏**：
- 目标：在规定时间内点击目标
- 操作：鼠标点击或触摸屏幕
- 得分：根据准确度和速度计分

**猜数字游戏**：
- 目标：猜出系统随机生成的数字
- 操作：输入数字并提交
- 提示：系统会给出"大了"或"小了"的提示

#### 4.3 游戏控制

- **开始游戏**：点击"开始"按钮
- **暂停游戏**：点击"暂停"按钮
- **重新开始**：点击"重置"按钮
- **查看得分**：在游戏界面右侧查看当前得分

### 5. 自定义和扩展

#### 5.1 自定义工具

开发者可以创建自定义绘画工具：

```dart
class MyCustomTool extends ToolPlugin {
  @override
  String get id => 'my_tool';
  
  @override
  String get name => '我的工具';
  
  @override
  void onCanvasEvent(CanvasEvent event) {
    // 自定义绘画逻辑
  }
}
```

#### 5.2 自定义游戏

开发者可以添加自定义游戏：

```dart
class MyCustomGame extends SimpleGame {
  @override
  String get id => 'my_game';
  
  @override
  String get name => '我的游戏';

  void start() {
    // 游戏开始逻辑
  }

```

## 快捷键

### 通用快捷键
- `Ctrl+N`：新建项目
- `Ctrl+O`：打开项目
- `Ctrl+S`：保存项目
- `Ctrl+Z`：撤销
- `Ctrl+Y`：重做
- `Ctrl+G`：切换到游戏模式
- `Ctrl+D`：切换到设计模式

### 绘画快捷键
- `B`：选择画笔工具
- `P`：选择铅笔工具
- `E`：选择橡皮擦
- `R`：选择矩形工具
- `C`：选择圆形工具
- `L`：选择线条工具
- `[`：减小画笔大小
- `]`：增大画笔大小

### 视图快捷键
- `Ctrl++`：放大画布
- `Ctrl+-`：缩小画布
- `Ctrl+0`：重置缩放
- `Space+拖拽`：平移画布

## 设置和配置

### 1. 界面设置

**主题选择**：
- 浅色主题：适合白天使用
- 深色主题：适合夜间使用
- 自动主题：根据系统设置自动切换

**布局设置**：
- 标准布局：传统的工具栏+画布+面板布局
- 紧凑布局：适合小屏幕设备
- 自定义布局：用户可自由调整面板位置

### 2. 性能设置

**渲染质量**：
- 高质量：最佳视觉效果，消耗更多资源
- 标准质量：平衡性能和质量
- 高性能：优先性能，适合低端设备

**自动保存**：
- 启用/禁用自动保存
- 设置自动保存间隔（1-30分钟）
- 设置最大备份数量

### 3. 输入设置

**鼠标设置**：
- 鼠标灵敏度调整
- 右键菜单配置
- 滚轮缩放设置

**触摸设置**：
- 多点触控支持
- 手势识别
- 压感支持

## 故障排除

### 常见问题

**Q: 画布无法绘制**
A: 检查是否选择了正确的工具，确保画布处于可编辑状态

**Q: 项目保存失败**
A: 检查磁盘空间是否充足，确保有写入权限

**Q: 游戏无法启动**
A: 尝试重新加载游戏，检查是否有错误提示

**Q: 界面显示异常**
A: 尝试重置界面布局，或重启应用

### 性能优化建议

1. **定期清理**：删除不需要的项目和临时文件
2. **合理使用图层**：避免创建过多图层
3. **适当的画布大小**：根据需要设置合适的画布尺寸
4. **关闭不必要的功能**：如不需要可关闭自动保存等功能
